LANGKAH - LANGKAH KONFIGURASI SAMBA SERVER DEBIAN :

Penjelasan :

Samba adalah aplikasi server yang memungkinkan Linux berbagi file dan printer dengan PC client Windows dan Linux.

  1. Instal paket samba
    # apt install samba
  2. Masuk ke home
    # cd /home
  3. Buat direktori baru, sesuka kita. Contoh saya membuat direktori myfile dan software
    # mkdir myfile
    # mkdir software
  4. Beri permission untuk direktori yang kita buat
    # chmod 777 myfile/
    # chmod 777 software/
  5. Tambahkan user, misalnya saya akan menambahkan user salman
    # useradd salman
  6. Beri password untuk user kita
    # smbpasswd -a salman
  7. Konfigurasi file samba
    # nano /etc/samba/smb.conf
    dibagian ini kalian ke bagian paling bawah kemudian ketikan 
    
[myfile] 
path	= /home/myfile
valid user = salman
browseable = yes
writeable = yes
guest ok = no 
      
[software]
path	= /home/software
valid user = salman
browseable = yes
writeable = no
guest ok = yes
    Untuk yang didalam kurung siku sesuai dengan nama direktori kita
  8. Kemudian restart
    # service smbd restart
  9. Tekan pada keyboard , windows + r . kemudian ketikan ip debian kita contoh disini ip debian saya \\172.17.15.3
