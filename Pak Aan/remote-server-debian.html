LANGKAH-LANGKAH MELAKUKAN REMOTE SERVER DEBIAN :

  1. Untuk settingan network adapternya seperti gambar
  2. Setting Ip debian dan Ip VM1 menjadi satu network. Contoh Ip debian 192.168.1.1 dan untuk Ip VM1 192.168.1.124
  3. Untuk default gateway-nya masukan Ip debian yang sudah kita buat yaitu 192.168.1.1
  4. Setting ip laptop kita
  5. Instal paket ssh server di debian, ketikan perintah : apt-get install openssh-server
  6. Kemudian ketikan perintah : nano /etc/ssh/sshd_config Untuk keluar dari sesi ini tekan : ctrl + o (untuk save) kemudian tekan Enter, tekan ctrl + x (untuk keluar)
  7. Restart ssh, ketikan perintah : service ssh restart
  8. Masuk ke aplikasi PUTTY, pada bagian Host Name masukan ip address debian yang sudah kita setting di bagian interfaces. Kemudian klik open
  9. login as : root, password : password debian yang kita buat
