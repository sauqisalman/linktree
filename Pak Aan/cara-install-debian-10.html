CARA INSTAL DEBIAN 10 di VMWARE

  1. Pastikan kita sudah mempunyai file iso debian 10, jika belum klik disini
  2. Buka vmware, kemudian klik create a new virtual machine
  3. kemudian klik typical (recommended), next
  4. Pilih i will install the operating system later, kemudian next
  5. Disini berikan nama virtual machine kalian (bebas) dan untuk lokasi penyimpanannya (bebas), kemudian klik next
  6. Disini pilih yang store virtual disk as a single file, untuk maximun disk sizenya bisa desesuaikan. Kemudian next
  7. Masuk ke settingan customize hardware, tambahkan network adapter dan setting sebagai host-only. Masukan juga file iso debian
  8. Kemudian jalankan Debian-nya
  9. Untuk install pada bagian awal seperti biasa, tinggal sesuaikan saja
  10. Pada bagian ini kita akan memilih network adapater primary-nya pilih ens33, kemudian next
  11. Pada bagian ini isi bebas
  12. Ini kosongkan saja
  13. Untuk bagian ini, disini kita akan mengatur partisi untuk debian kita. Sebaiknya kita pilih yang bagian guided-use entire disk saja jika belum paham tentang cara membagi partisi di debian. Pada bagian ini pilih yang guided-use entire disk dan tinggal next next saja dan pada bagian write changes pilih yes
  14. Ini pilih no
  15. Ini pilih yes
  16. Ini pilih /dev/sda
  17. Ini pilih continue, selesai deh..
