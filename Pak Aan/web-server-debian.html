LANGKAH - LANGKAH KONFIGURASI WEB SERVER DEBIAN :

  1. Instal apache2 di debian, ketikan perintah : apt install apache2
  2. Setting ip laptop kita dulu. Untuk preferred dns server kita masukan ip debian kita.
  3. Coba ketikan ip debian kita di browser, jika berhasil akan ada tampilan seperti gambar. Tandanya kita telah berhasil menginstal apache2 di debian kita.
  4. Sekarang kita akan membuat web server untuk, contoh tkj.com dan tjkt.com
  5. Masuk ke direktori html apache. Jika kita ketikan perintah ls maka akan keluar index.html, kita akan buar folder tkj yang berisi index.html 
    root@debian:cd /var/www/html
  6. Buat folder tkj 
    root@debian:/var/www/html# mkdir tkj

root@debian:/var/www/html# cd tkj

root@debian:/var/www/html/tkj# nano index.html
  7. Kita buat file html sederhana untuk website tkj
  8. Kita masuk ke direktori sites-available, kemudian kita copy 000-default.conf untuk namanya menjadi tkj.conf 
    root@debian:cd /etc/apache2/sites-available

root@debian:/etc/apache2/sites-available# cp 000-default.conf tkj.conf
  9. Kita masuk ke tkj.conf, kemudian edit di bagian yang ditandai merah, jangan lupa untuk di save. 
    root@debian:/etc/apache2/sites-available# nano tkj.conf
  10. Setelah itu restart apache2 
    root@debian:/etc/apache2/sites-available# systemctl restart apache2
  11. Setelah itu a2ensite tkj.conf 
    root@debian:/etc/apache2/sites-available# a2ensite tkj.conf
  12. Setelah itu restart lagi apache2 
    root@debian:/etc/apache2/sites-available# systemctl restart apache2
  13. Lakukan hal yang sama jika ingin membuat tjkt.com mulai dari bagian html untuk tjkt dan untuk sites-availablenya jangan lupa buat juga untuk tjkt.conf
