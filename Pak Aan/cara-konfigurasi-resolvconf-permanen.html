Masalah yang terjadi adalah file resolv.conf akan berubah kembali seperti semula ketika debian di restart

  1. Instal paket resolvconf 
    root@debian: apt install resolvconf
  2. Aktifkan dan jalankan resolvconf 
    root@debian: systemctl enable resolvconf
root@debian: systemctl start resolvconf
root@debian: systemctl status resolvconf
  3. Masukan ip address nameserver di dalam file head konfigurasi resolvconf. ip address dns server di letakan di paling atas kemudian selanjutnya untuk nameserver ip google 
    root@debian: nano /etc/resolvconf/resolv.conf.d/head
  4. Jika ingin merubah atau menambah ip address nameserver, ubah dibagian file head dan jangan lupa untuk melakukan update resolvconf 
    root@debian: resolvconf -u
