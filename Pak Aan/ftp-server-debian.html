LANGKAH - LANGKAH KONFIGURASI FTP SERVER DEBIAN (vsftpd) :

  1. Install vsftpd, ketikan perintah : apt install vsftpd
  2. Ketikan perintah : nano /etc/vsftpd.conf . Edit yang bertanda merah.
  3. Setting IP laptop kita agar satu network dengan IP Debian
  4. Setelah mengedit lakukan restart, ketikan perintah : /etc/init.d/vsftpd restart, kemudian tekan enter.
  5. Setelah itu buat user untuk log on ke ftp nantinya, ketikan perintah : adduser (namauser)
  6. Kita akan daftarkan user yang boleh log on ke ftp, ketikan perintah : nano /etc/vsftpd.chroot_list contohnya disini saya akan memberi akses untuk user sauqisalman, maka kita ketikan sauqisalman atau nama user yang akan didaftarkan
  7. Setelah mendaftarkan user yang bisa log on ftp, jangan lupa untuk restart kembali, ketikan perintah : /etc/init.head/vsftpd restart
  8. Sekarang kita coba untuk mengakses ftp server debian yang sudah kita buat, kita buka file eksplorer pc kita, dibagian yang bertanda merah ketikan : ftp://172.17.15.3. Maka akan muncul kotak seperti gambar, isikan user yang sudah kita buat. Disini saya menggunakan user sauqisalman dan masukan password yang sudah kita buat kemudian tekan Log On. kondisinya masih kosong tidak ada file atau folder
  9. Sekarang kita akan buat folder baru, pertama-tama kita buka cmd kemudian ketikan perintah : ftp 172.17.15.3. Akan diminta memasukan user, masukan user yang telah kita daftarkan disini saya akan masuk menggunakan user sauqisalman. Kemudian ketikan perintah mkdir (nama folder yang ingin kalian buat), di sini saya membuat folder dengan nama software
  10. Kita akan isi folder tersebut, kalian bisa memasukan file aplikasi, file word, foto, dan sebagainya. Kalian buka aplikasi FileZilla. Untuk Host ketikan IP ftp debian kita, untuk username dan password masukan user yang sudah kita daftarkan dan untuk port 21 . Kemudian klik Quickconnect Tekan OK. Contoh disini saya akan memasukan aplikasi winbox ke folder software, pilih aplikasi winbox kemudian drag and drop ke bagian kotak merah.
  11. Sekarang kita lihat lagi di file eksplorer. Sudah ada folder software yang sudah kita buat tadi dan jika kita masuk ke foldernya sudah ada aplikasi winbox yang sudah kita masukan menggunakan FileZilla.
  12. udah deh..

LANGKAH - LANGKAH KONFIGURASI FTP SERVER DEBIAN (proftpd) :

  1. Instal proftpd, ketikan perintah : apt install proftpd
  2. Ketikan perintah : nano /etc/proftpd/proftpd.conf ubah ipv6 on menjadi off di sebelah anonymous masukan lokasi direktori file ftp kita
  3. masuk ke direktori home, ketikan perintah : cd /home , kemudian buat direktori baru, contoh disini saya membuat direktori dengan nama file-ftp. Ketikan perintah : mkdir file-ftp
  4. Ketikan perintah : chmod 777 file-ftp/
