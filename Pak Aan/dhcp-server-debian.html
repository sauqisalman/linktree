LANGKAH - LANGKAH KONFIGURASI DHCP SERVER DEBIAN :

  1. Instal dhcp server, ketikan perintah : apt install isc-dhcp-server
  2. Konfigurasi dhcp server, ketikan perintah : nano /etc/dhcp/dhcpd.conf. Edit pada bagian seperti gambar

    Penjelasan :

    • subnet berisi net id ip kalian, contoh ip yang digunakan 192.168.28.1 maka untuk subnetnya 192.168.28.0
    • netmask yaitu subnetmask yang kita gunakan, contoh disini kita menggunakan /28 maka untuk netmasknya 255.255.255.240
    • range adalah jumlah host yang ingin kita berikan ip dhcp, contoh disini kita menggunakan /28 maka untuk jumlah hostnya adalah 14 host jadi untuk ip rangenya adalah 192.168.28.2 (bukan 192.168.28.1 karena itu merupakan ip debian) sampai 192.168.28.14
    • option domain-namer-server diisi dengan ip debian kita, contoh disini ip debian kita 192.168.28.1
    • option domain-name diisi dengan dns yang sudah kita buat pada dns server debian kita, contoh disini saya mengisinya dengan tkj.com
    • option routers diisi dengan ip debian kita, contoh disini ip debian kita 192.168.28.1
    • option brodcast-address diisi dengan ip broadcast dari ip yang kita gunakan, contoh disini ip kita 192.168.28.1/28 maka untuk ip broadcastnya yaitu 192.168.28.15
  3. Kemudian ketikan perintah : nano /etc/default/isc-dhcp-server, isi pada bagian INTERFACESv4 sesuaikan pada interfaces yang ada pada kalian
  4. Setelah itu masuk ke virtual network editor, disini kita changes settings pada bagian host-only vmnet1, kita unchecklis pada bagian use local DHCP service
  5. Masuk ke setingan network adapter pc kita, lihat pada bagian vmnet1
