LANGKAH-LANGKAH KONFIGURASI MAIL SERVER DEBIAN

sumber : https://youtu.be/JlWO4ItWoGY

  1. Instal mariadb-server
    # apt install mariadb-server -y
  2. Konfigurasi database supaya bisa diakses oleh user root
    # mariadb -u root
    kemudian ikuti ketikan seperti gambar
  3. Instal dovecot
    # apt install dovecot-imapd -y
  4. Lanjut ke konfigurasi dovecotnya, kita masuk ke direktori dovecot
    # cd /etc/dovecot/conf.d
  5. Kemudian masuk ke 10-mail.conf
    # nano 10-mail.conf
    kemudian ikuti seperti gambar
  6. Lanjut instal postfix
    # apt install postfix -y
    pada bagian ini kita pilih local only saja kemudian untuk bagian system mail name kita isikan dengan domain yang sudah kita buat
  7. Kemudian kita tentukan mail direktori mail servernya
    # maildirmake.dovecot /etc/skel/Maildir
  8. Lanjut seting postfix-nya
    # nano /etc/postfix/main.cf
    kebagian paling bawah kemudian pada baris baru ketikan seperti gambar
  9. Kita konfigurasi ulang postfix-nya
    # dpkg-reconfigure postfix bagian other destinations to accept .... kita isikan dengan domain kita dan localhost, seperti gambar untuk locat networks kita isikan dengan network dari IP yang kita gunakan dan masukan juga 0.0.0.0/0 enter enter pilih yang ipv4
  10. Kemudian install roundcube
    # apt install roundcube -y
  11. kemudian kita konfigurasi roundcube
    # dpkg-reconfigure roundcube
    isikan dengan domain yang sudah kita buat pilih yang TCP/IP enter enter enter enter kosongin aja enter pastikan sudah dicentang dua-dua nya enter
  12. Masuk ke direktori roundcube
    # cd /etc/roundcube
  13. Kemudian masuk ke
    # nano config.inc.php
    kemudian kosongkan bagian yang diberi tanda panah
  14. Kemudian kita buat 2 user baru
    # adduser .....
  15. Kemudian kita reboot debian kita
  16. Kita seting webserver untuk roundcube-nya
    # cd /etc/apache2/sites-available
  17. Copy file 000-default.conf
    # cp 000-default.conf webmail.conf
  18. Kemudian kita masuk ke webmail.conf yang sudah kita buat tadi
    # nano webmail.conf
    ketikan seperti gambar
  19. Jangan lupa untuk kita aktifkan
    # a2ensite
  20. dan jangan lupa untuk restart apache2 nya
    # systemctl restart apache2
  21. Kemudian kita pergi ke browser ketikan webmail.tkj.id dan masukan user yang sudah dibuat
