KONFIGURASI DHCP SERVER MIKROTIK

Penjelasan :

DHCP atau Dynamic Host Configuration Protocol adalah salah satu service yang dapat membuat perangkat membagikan IP Address secara otomatis pada host dalam sebuah jaringan. Tidak hanya IP Address saja, DHCP juga dapat membagikan informasi default gateway, DNS, dan netmask network.

Langkah-Langkah :

Disini kita akan memberikan IP kepada PC 1, 2, dan 3 secara dhcp(secara otomatis).

  1. Buka router, kemudian ketikan perintah : interface bridge add (artinya kita akan menambahkan bridge di router kita). Setelah itu kita ketikan perintah : interface bridge print (kita akan melihat apakah bridge yang sudah kita buat sudah ada atau belum).
  2. Setelah itu kita masukan port ether1, ether2, dan ether3 ke dalam bridge. Ketikan perintah : interface bridge port add, untuk interface kita masukan satu persatu dan untuk bridge kita masukan bridge1.
  3. Selanjutnya kita beri IP bridge1, ketikan perintah : ip address add address=192.168.30.1/29 interface=bridge1. Kemudian ketikan perintah : ip address print (untuk melihat apakah IP yang kita berikan sudah masuk atau belum).
  4. Kita akan membuat dhcp-server untuk bridge1 agar bridge1 bisa memberikan IP ke PC1, 2, dan 3 secara otomatis. Ketikan perintah : ip dhcp-server setup, kemudian untuk interface : bridge1.
  5. Ketikan perintah : dhcp, pada PC1, 2, dan 3. Maka secara otomatis PC akan mendapatkan alamat IP.
