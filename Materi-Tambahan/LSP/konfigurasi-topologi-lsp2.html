Langkah-Langkah Konfigurasi Topologi LSP

  • KONFIGURASI RB MASTER
  • Yang di setting di RB Master
    1. Masukan sumber internet
    2. Tambahkan dhcp-client untuk sumber internet
    3. Setting dns untuk sumber internet
    4. Tambahkan firewall nat untuk sumber internet
    5. Tambahkan vlan yang dibutuhkan
    6. Beri ip untuk vlan
    7. Tambahkan dhcp-server untuk vlan
    8. Tambahkan firewall filter
  1. Reset router
  2. Masukan sumber internet. Bisa menggunakan wlan atau kabel ethernet
  3. Tambahkan dhcp-client sumber internet
  4. Aktifkan allow remote request dns
  5. Tambahkan firewall nat untuk sumber internet, disini kita menambahkan untuk wlan1
  6. Coba untuk tes ping 8.8.8.8
  7. Tambahkan vlan10 dan vlan20 pada interface ether1
  8. Tambahkan ip address untuk vlan10 dan vlan20
    vlan10 = 10.10.10.1/28
    vlan20 = 10.10.10.17/29
  9. Buat dhcp-server untuk vlan10 dan vlan20
  10. Tambahkan firewall filter
  • KONFIGURASI RB SWITCH
  • Yang di konfigurasi di RB Switch
    1. Buat bridge
    2. Setting vlan-filtering menjadi yes
    3. Tambahkan bridge port untuk ehter yang terhubung
    4. Tambahkan bridge vlan
  1. Reset router
  2. Buat bridge
  3. Setting vlan-filtering menjadi yes
  4. Tambahkan bridge port untuk ether1, ether2, dan ether3
    untuk ether2 dan ether3 tambahkan pvid
  5. Tambahkan bridge vlan
  • SETTINGAN UNTUK PC CLIENT
  1. Buka network connection di pc kemudian cek port yang terhubung dengan router, apakah sudah mendapatkan ip Jika belum dapat ip, disable kemudian enable port ethernet
  • Menambahkan VLAN-Management Berfungsi untuk melakukan remote router
  1. Di rb master buat 1 vlan pembantu (vlan99, dengan ip misalkan 192.168.99.1/29), di ether yang sama dengan vlan10 dan vlan20
  2. di rb switch buat juga vlan99, dengan ip satu network dengan ip 99 di rb master misalkan 192.168.99.2/29 letak vlan99 di rb swith terletak pada interface bridge vlan
  • Settingan RB Master
  1. Tambahkan vlan management, contoh vlan99 di interface yang sama dengan vlan10 dan vlan20
  2. Beri ip address vlan99, 192.168.99.1/29
  3. Setting dhcp-server untuk vlan99
  • Settingan untuk RB Switch
  1. Tambahkan vlan99 untuk interface bridge-vlan
  2. Tambahkan bridge vlan untuk vlan-id 99, tagged bridge dan ether trunking
  3. Tambahkan ip dhcp-client untuk interface vlan99
  • Cara melakukan remote router
  1. Untuk meremot antar mikrotik colokan kabel ke sumber vlan di rb switch
  2. Untuk meremot rb master seharusnya sudah bisa
  3. Jika ingin meremot rb switch masukan ip yang sudah di dapat dari dhcp-client
