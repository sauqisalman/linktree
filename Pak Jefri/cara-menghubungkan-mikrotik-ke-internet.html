1. Syarat, untuk versi GNS3 dan GNS VM sama
  2. Untuk setingan network connection seperti gambar
  3. Kemudian kita buka setingan ip di pc kita, kita shared wifi yang kita gunakan. Ceklis di bagian kotak merah, terus kotak di bawahnya pilih untuk vmnet1
  4. Kita pergi ke aplikasi GNS3, kemudian kita masuk ke pilihan edit --> preferences...
    klik GNS3 VM --> ceklis enable the GNS3 VM --> arahkan VM name ke GNS3
    kemudian klik apply --> oke
  5. Tunggu sampai server summary pada gns3 menjadi hijau
  6. Buat topologi sederhana seperti gambar
  7. Tambahkan mikrotik baru, tapi untuk server typenya seperti gambar
  8. Buka mikrotik, kemudian ketikan perintah : ip add print. Seharusnya sudah mendapatkan ip pada ether1
  9. Kemudian ketikan perintah : ip dhcp-client print. Seharusnya statusnya sudah bound
  10. Coba untuk tes ping mikrotik ke 8.8.8.8 dan google.com
  11. Kita tambahkan ip untuk ether2(port ether yang terhubung ke pc)
  12. Kemudian tambahkan ip firewall nat pada mikrotik
  13. Buka pc, kemudian tambahkan ip yang satu segment dengan ip pada ether2 di mikrotik dan untuk gatewaynya masukan ip ether2 mikrotik. Tambahkan juga ip dns
  14. Sekarang coba tes ping 8.8.8.8 pada pc
